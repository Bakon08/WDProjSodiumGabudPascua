Page Guide

How this page works

Date Range

How it works: Every metric on this page responds to the selected range. How to use it: Start with Month for a balanced view, then compare with Week and 3-Month to spot momentum changes.

Trend Chart

How it works: The line chart groups completed tasks into weekly buckets and computes completion rate. How to use it: Watch for dips and then check Planner to reduce overload in those weeks.

Time Analysis

How it works: Category bars estimate effort and heatmaps count when tasks were completed. How to use it: Schedule high-priority tasks during your strongest day/hour blocks.

Goal Progress

How it works: Goal progress is based on linked Planner tasks, with fallback to goal completion state when no links exist. How to use it: Link tasks to goals in Planner so these bars stay accurate and actionable.

Streaks and Records

How it works: Streaks count consecutive days with at least one completed task, and records track all-time bests. How to use it: Complete at least one meaningful task daily to keep streaks alive.

How Data Is Computed

How it works: Data comes from localStorage task and goal records, including completion timestamps. How to use it: Keep task progress updated in Planner and goal links clean so analytics reflect your real workflow.
